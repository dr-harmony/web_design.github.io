Weather Data

The purpose of this project was to compare weather of a list of randomly selected cities around the world. To do this, data from the OpenWeatherMap API was used to assemble a dataset of over 500 cities. Data was then converted to a pandas dataframe for downstream analysis.

After compiling the list of cities and weather data in a pandas dataframe. Matplotlib was used to produce plots comparing different aspects of the dataset (examples available in the "Visualizations" tab) from the raw data (available in the "Data" tab). In 3 of these plots, linear regression analysis was used to determine if there was a correlation between the parameters plotted. All plots can be visualized at once in the "Comparisons" tab. Descriptions of each graph are available in each graphs respective link.
